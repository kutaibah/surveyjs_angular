# SurveyjsAngularCli

This project was built over  [surveyjs_angular_cli](https://github.com/surveyjs/surveyjs_angular_cli)


## Steps to Deploy

 1. **Clone the project**  
	```
	git clone https://github.com/kutaibah/surveyjs_angular
	```

 2. **Install node dependencies**
 	```
	node install
	```
 3. **Run the Project**
 	```
	 ng serve
	```
